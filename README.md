# Gojo Infinity Mod
A Minecraft Forge 1.20.1 mod that recreates Satoru Gojo's Infinity technique from Jujutsu Kaisen.
The goal of this project is to simulate the concept of Infinity using custom entity physics and Forge event systems.

## Features

### Infinity Ability
- Toggle Infinity using a custom keybind
- Prevents entities from reaching the player
- Creates a slowing effect as objects approach

### Entity Manipulation
- Hostile mobs are slowed near the player
- Entities cannot pass through the Infinity barrier
- Close-range objects are pushed away

### Projectile Protection
- Projectiles are slowed before reaching the player
- Arrows and other moving entities are affected

### Damage Protection
- Blocks harmful effects while Infinity is active
- Protects against attacks and explosions

## Controls
| Key | Action |
|---|---|
| G | Toggle Infinity |

## Technologies
- Java 17
- Minecraft Forge 1.20.1
- Gradle

## Installation
1. Install Minecraft Forge 1.20.1
2. Download the mod `.jar`
3. Place it inside the Minecraft `mods` folder
4. Launch Minecraft

## Development
This project was created as a Java programming and Minecraft modding study project.
Main concepts explored:
- Forge Event System
- Entity manipulation
- Custom physics behavior
- Client key bindings
